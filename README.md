# MSI-B450m-MORTAR EFI

> Hackintosh EFI for MSI B460M MORTAR + i5-10400
>
> 无独显版本

## 版本

Mac 版本：11.1

Opencore 版本：0.6.5

更新日期：2021-2-2

## 配置清单

| 类型 |          型号           |
| ---- | :---------------------: |
| CPU  |        i5-10400         |
| 主板 |    微星 B460M MORTAR    |
| SSD  | WD SN550 500G 蓝盘 NVME |
| 内存 |  宇瞻 黑豹 3000 16G*2   |
| 网卡 |       BCM94360CS2       |
## BIOS设置

1. **关闭cfg lock（必须）**
2. **关闭Intel VT-D Tech**
3. **关闭IGD Multi-Monitor**
4. **关闭Intel Virtualization Tech**

## 注意事项

1. **登陆Apple ID前先检查序列号等信息，避免与他人重复**
